# Analysis pipelines

This section provides guidelines and example workflows for analyzing experimental data collected in the lab.

The goal is to offer **reproducible, standardized pipelines** for common data types, helping researchers move efficiently from raw data to results.

---

## What you will find here

- Step-by-step analysis workflows  
- Recommended tools and software packages  
- Example scripts and code snippets  
- Best practices for reproducible research  

---

## Available pipelines

Use the sidebar to navigate to specific analysis pipelines, such as:

- EEG analysis  
- Eye tracking analysis  
- Neurostimulation analysis  

---

## General principles

When analyzing data in the lab, aim to:

- Keep raw data unchanged  
- Work with clearly separated processing steps  
- Document all analysis decisions    
- Prefer reproducible scripts over manual steps
