# OpenSesame

OpenSesame (Mathôt et al., 2012) is an **open-source experiment builder** that is frequently used at
Experimental Psychology for running behavioral and psychophysiological experiments. It allows
researchers to design experiments using a graphical interface, Python scripting, or a combination
of both.

OpenSesame provides **extensive tutorials and examples** that are suitable for both beginners and
more advanced users. This page provides a short practical introduction to getting started with
OpenSesame and points to the most relevant documentation.

---

## Installing OpenSesame

OpenSesame can be downloaded from the official website:  
[https://osdoc.cogsci.nl/](https://osdoc.cogsci.nl/)

Installation instructions differ slightly by operating system and are described in detail on the
website. In general, the main steps are:

1. Download the latest stable version of OpenSesame for your operating system.
2. Follow the installation instructions for your platform.
3. After installation, start OpenSesame and verify that it opens without errors.

On lab computers, OpenSesame may already be installed. If you are unsure which version is
available, check with lab support before installing or updating software.

---

## Creating a basic experiment

A typical workflow when creating a new OpenSesame experiment is:

1. Create a new experiment and select the appropriate backend.
2. Define experimental variables in a loop table  
   (e.g. stimulus, condition, correct response).
3. Use sketchpads to present stimuli.
4. Use response items to collect participant input.
5. Test the experiment using the **Run** button.
6. Inspect the output file to verify that all relevant variables are logged correctly.

---

## Using OpenSesame with lab equipment

OpenSesame is commonly used in combination with:

- Eye trackers (e.g. Eyelink)
- EEG systems (e.g. sending triggers)
- External response devices

## SigmundAI research assistant

SigmundAI ([https://sigmundai.eu](https://sigmundai.eu)) is an AI-based research assistant designed
to help with OpenSesame. SigmundAI can be used directly from within OpenSesame by installing it
as an extension.

More information about SigmundAI can be found in the OpenSesame documentation:  
[https://osdoc.cogsci.nl/4.0/manual/sigmund/](https://osdoc.cogsci.nl/4.0/manual/sigmund/)
