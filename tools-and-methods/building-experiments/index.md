# Programming experiments

Experiments in the Experimental Psychology department are typically programmed using **OpenSesame**, an open-source experiment builder designed for behavioral and psychophysiological research.

OpenSesame allows researchers to design experiments using a graphical interface while still providing the flexibility of Python scripting when needed. It integrates well with common laboratory equipment such as eye trackers and EEG systems.

Because of its flexibility and compatibility with our laboratory setups, **OpenSesame is the standard tool used in the department for programming experiments**.

---

## Getting started

If you are programming a new experiment, the recommended workflow is:

1. Design and program the experiment in **OpenSesame**
2. Test the experiment locally to ensure that timing and responses work correctly
3. Test the experiment on the **lab computer** with the relevant hardware connected (e.g., eye tracker or EEG)
4. Verify that all variables and responses are correctly stored in the output data file

Once the experiment runs reliably, it can be used for pilot testing and data collection.
