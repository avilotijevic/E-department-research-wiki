# Computing servers

This section provides information about the computing servers available for data analysis within the department.

Our servers are designed to handle computationally intensive tasks such as EEG analysis, machine learning, and large-scale data processing.

---

## Available servers

### Fox

**[Fox](../computing-servers/fox.md)** is the Experimental Psychology department's primary server used for data analysis. It provides high-performance computing resources, including GPU support for machine learning workflows.


### Hábrók
**[Hábrók](../computing-servers/habrok.md)** is the University of Groningen’s high-performance computing (HPC) cluster—a very powerful collection of interconnected computers used for demanding research tasks.


## When should you use a server?

You should use a computing server when:

- your analysis is too slow on a local machine  
- you need GPU acceleration (e.g., deep learning)  
- you are working with large datasets  
- you want to run long analyses without interruption  

---

## Getting access

Access to computing servers is managed by the system administrators.  
Please contact them to request an account.
